Descripción del Juego.

Inicio: El juego comienza y se crea la ventana principal.

Inicialización: Se inicializan las variables del juego, como el tablero y el contador de clics.

Creación de Botones: Se crean los botones que representan las casillas del tablero y el botón de reinicio.

Mostrar Ventana: La ventana del juego se muestra en la pantalla.

Función jugar(indice):

  • Se verifica si la casilla seleccionada está vacía.
  • Si está vacía, se incrementa el contador de clics.
  • Dependiendo del número de clics, se asigna "O" o "X" a la casilla.
  • Se actualiza el texto del botón correspondiente.

Verificación de Ganador:

  • Se llama a la función verificar_ganador().
  • Se comprueba si hay un ganador o un empate.
  • Si hay un ganador, se muestra un mensaje y se reinicia el juego.

Empate: Si no hay ganador y todas las casillas están ocupadas, se muestra un mensaje de empate y se pregunta si se desea reiniciar.

Reinicio:

  • Si se decide reiniciar, la función reiniciar() limpia el tablero y restablece el contador de clics.
  • Se actualizan los botones para reflejar el estado inicial.

Fin: El juego vuelve a estar listo para jugar.
